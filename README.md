# RevPlay – Music Streaming Platform 

RevPlay is a full-stack music streaming platform designed to provide users with an engaging music listening experience while enabling artists to manage and publish their music.
This repository contains the **backend services built using Java Spring Boot and MySQL**, responsible for handling authentication, music management, playlists, analytics, and platform features.

## Frontend Repository

The frontend of RevPlay is developed using **Angular**.

Frontend Repository:
https://github.com/yourusername/revplay-frontend

## Key Features

### User Features

* Secure user registration and login with **JWT-based authentication**
* Advanced **music search and discovery**
* Integrated **music player** with play, pause, skip, shuffle, and repeat
* Create, update, reorder, and delete **custom playlists**
* **Favorites management** for quick access to liked songs
* **Listening history tracking** for personalized experience
* Dynamic **mix playlists** based on user activity

### Artist Features

* Artist profile creation and management
* Upload and manage songs and albums
* Control music visibility and metadata
* Access **analytics insights** such as total plays and engagement metrics

### Platform Features

* **Premium features** for enhanced user experience
* **Advertisements integration** for monetization
* **Analytics dashboards** showing listening trends and platform activity
* Optimized **REST APIs** for seamless frontend integration
* Secure and scalable backend architecture

## Architecture

RevPlay follows a **full-stack architecture** where the Angular frontend communicates with the Spring Boot backend through REST APIs, while MySQL manages structured data such as users, songs, playlists, and analytics.

## Project Goal

The goal of RevPlay is to simulate a **modern music streaming platform**, demonstrating full-stack development, secure authentication, scalable backend design, and interactive user features.
