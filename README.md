# RevPlay – Full Stack Music Streaming Platform

RevPlay is a full-stack music streaming web application designed to deliver an engaging music listening experience while enabling artists to upload and manage their content.
This repository contains the **backend implementation built using Java Spring Boot and MySQL**, providing secure APIs, business logic, and platform services.

## Frontend Repository

The frontend application is developed using **Angular**.

Frontend Repository:
https://github.com/Conginzant-Java-FSE/RevPlay-UI

## Key Features

### User Features

* Secure **user registration and login** with authentication and role-based access control
* Advanced **music search and discovery**
* Integrated **music player** with play, pause, skip, shuffle, repeat, and volume control
* Create, edit, reorder, and delete **custom playlists**
* **Favorites management** for quick access to liked songs
* **Listening history tracking** to monitor user activity
* **Dynamic mix playlists** generated based on listening patterns

### Artist Features

* **Artist profile creation and management**
* Upload and manage **songs and albums**
* Update song metadata and visibility settings
* Access **analytics dashboards** with song plays, favorites, and listener engagement

### Platform Features

* **Premium features** including ad-free listening and enhanced functionality
* **Advertisements integration** for platform monetization
* **Analytics and insights dashboards** to visualize listening trends
* Optimized **RESTful APIs** for seamless frontend-backend communication
* Secure and scalable **Spring Boot backend architecture**

## Architecture

RevPlay follows a **full-stack architecture** where the Angular frontend communicates with the Spring Boot backend through REST APIs.
MySQL is used for managing structured data including users, songs, playlists, listening history, and analytics data.

## Project Goal

The goal of RevPlay is to simulate a **modern music streaming platform similar to Spotify**, demonstrating full-stack development, scalable backend architecture, secure authentication, and interactive user features.
